## Setup

```sh
git clone https://github.com/GPT-Engineer-App-Dev/sollenkroka-kayak-adventures.git
cd sollenkroka-kayak-adventures
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
